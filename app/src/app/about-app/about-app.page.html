SOBRE O APLICATIVO

FLASHCARDS

O aplicativo de Flashcards foi desenvolvido para auxiliar a comunicação e educação de crianças com autismo.

O aplicativo foi desenvolvido como Projeto de Graduação do aluno Alberto André Aguiar Neto, do curso de Sistemas para Internet da Fatec São Roque, utilizando a linguagem de programação JavaScript, especificamente o framework Ionic.

Este aplicativo é open source e você, sendo programador, pode contribuir com melhorias acessando o repositório https://github.com.br/albertoaguiar/flashcards.

Utilize este aplicativo da melhor forma, podendo assim facilitar a comunicação e educação de crianças com austismo.
